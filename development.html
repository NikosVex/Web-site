Разработка бота "Покажи питомца"

Бот написан на языке программирования Python с использованием следующих библиотек:
1. aiogram 3 - библиотека для разработки Telegram-ботов на Python.
2. SQLAlchemy 2 - библиотека для написания запросов к базе данных не на SQL, а сразу на Python.

Бот использует для хранения различной информации базу данных PostgreSQL, я думаю, это самая практичная и удобная БД для разработки.

Разработка бота "Покажи питомца" началась 8 декабря 2024 года и, по факту, была завершена спустя месяц, 8 января 2025 года, бот скоро будет полноценно выпущен на VPS-сервер для беспрерывной работы.

До создания бота с питомцами я изучал две эти библиотеки и писал более простых ботов, например бот с печньками с предсказаниями.

А этот сайт, на котором вы сейчас находитесь, был написан в рамках практической работы по информатике по HTML и CSS.
